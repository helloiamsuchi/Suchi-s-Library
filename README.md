# Suchi's Library📖

My private library collection to keep the records of the books I have read and my reviews on them.

**[Click here to see the demo!](https://helloiamsuchi.github.io/Suchi-s-Library/)**

## 📚 Functionality Flow
1. **Theme Toggle** → User selects Light or Dark theme → Bookshelf UI loads with chosen palette.  
2. **Bookshelf Layout** → Wooden bookshelf grid with placeholders for book covers.  
3. **Add Book (‘+’ Icon)** → Opens form → User uploads cover and details → Book added to shelf.  
4. **Book Interaction** → Tap on a book → Contextual menu opens with options:  
   - **View** → Opens new page → Left: enlarged book cover → Right: reviews & ratings.  
   - **Edit** → Editable form → Change cover, add star rating, genre, writer’s name, review → Save updates → Shelf refreshes.  
   - **Delete** → Removes book from shelf grid instantly.  
   - **Cancel** → Closes menu → Returns to bookshelf grid view.
